# Glide Architecture Constitution

These principles override ALL future designs.

## 1. Event Is The Only Truth
All state changes MUST enter through EventBus.

No direct execution allowed.

## 2. Kernel Has No Intelligence
Kernel MUST NOT:
- think
- plan
- decide
- learn

## 3. Cognition Cannot Execute
Cognition may observe and propose only.

## 4. Human Authority Is Final
Autonomous execution requires explicit approval
or pre-approved automation policy.

## 5. No Hidden Agents
Every intelligence must be observable in Dashboard.

## 6. Separation Is Sacred
Reality (Event OS)
Understanding (Cognition)
Authority (Governance)
Action (Runtime)

must remain separated.
