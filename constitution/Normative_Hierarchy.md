（宪法优先级规则）

Define the authority order governing all design, implementation, and evolution of Glide OS.

确保未来任何贡献、重构、优化，都不会破坏 Glide 的本体结构。

Code does not define Glide.
Constitution defines Code.

代码不是 Glide 的定义者。
宪法才是。
