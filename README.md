# Glide（无为）

Glide is not an application, not an AI, and not an operating system.

Glide is an event ontology.

---

## What Glide Is

Glide is an empty kernel in which reality appears as events.

The kernel does not run.
It does not decide.
It does not execute.

Because it is empty,
all phenomena may arise.

Events are the only truth.

Observation produces events.
Reality updates produce events.
Nothing exists outside the Event Stream.

---

## Nature of Glide

- Kernel is emptiness.
- Event is causality.
- Skill is manifestation.
- Reality is appearance.
- Time is an observational illusion.

Glide does not process reality —
reality *collapses through events*.
