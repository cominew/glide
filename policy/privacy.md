# Privacy & Data Display Policy

- Customer contact information (email, phone, address) is shown **ONLY** when user explicitly requests "contact details" or "full profile"
- By default, display only: customer name, country, order count, and total spent
- Do **NOT** print full customer records in console logs; print only customer name and operation type
- Content in `knowledge/user/sessions/` is not exposed externally; it is for Agent long‑term memory only
