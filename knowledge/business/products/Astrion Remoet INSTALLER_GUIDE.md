# Astrion Remote — Installer Guide

## Overview

This guide is intended for installers, integrators, and advanced users.

Astrion Remote acts as a smart control endpoint connected directly
to Home Assistant.

---

## Installation Workflow

1. Physical Installation
2. Network Configuration
3. Home Assistant Integration
4. UI Assignment
5. User Handover

---

## 1. Physical Installation

Recommended locations:

- Wall switch position
- Bedside controller
- Entrance scene panel
- Media room control

Power Requirements:
- Stable DC supply
- Continuous power recommended

---

## 2. Network Setup

Requirements:

- 2.4GHz Wi-Fi recommended
- DHCP enabled
- Same LAN as Home Assistant

Steps:

1. Enter setup mode
2. Select Wi-Fi network
3. Confirm IP assignment

---

## 3. Home Assistant Integration

Navigate:

Settings → Devices & Services

Add Astrion or allow auto discovery.

Assign:

- Areas
- Devices
- Scenes
- Dashboards

---

## 4. Interface Configuration

Recommended Layout:

Page 1 → Daily Controls  
Page 2 → Lighting  
Page 3 → Media  
Page 4 → Advanced Automation

---

## 5. Customer Handover

Installer checklist:

- Wi-Fi verified
- Scenes tested
- Firmware updated
- User trained
