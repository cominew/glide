# Astrion Remote

Smart Control. Simplified.

Astrion Remote is a customizable smart home controller built for seamless
integration with Home Assistant and modern automation ecosystems.

---

## ✨ Highlights

- Native Home Assistant Integration
- Fully Customizable Interface
- Scene & Automation Control
- Local-First Architecture
- OTA Firmware Updates
- Installer Friendly

---

## 📦 What Is Astrion?

Astrion Remote is designed to replace traditional wall panels, tablets,
and remotes with a unified smart control interface.

Control:

- Lighting
- Curtains
- Media Systems
- Climate
- Smart Scenes
- Dashboards

---

## 🚀 Quick Start

1. Power on Astrion Remote
2. Connect to Wi-Fi
3. Discover device in Home Assistant
4. Assign entities & scenes
5. Enjoy smart control
