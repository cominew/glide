# Troubleshooting

## Device Not Found

- Same network?
- Router isolation disabled?
- Restart Astrion

---

## Wi-Fi Problems

- Use 2.4GHz band
- Check signal strength
- Reconfigure network

---

## Slow Response

Possible causes:

- Cloud automations
- Network latency
- Overloaded HA instance

---

## Firmware Issues

Always update to latest firmware.
